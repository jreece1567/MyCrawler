# MyCrawler
Web Crawler

## Building
This is a Maven project. To build it, run the following command:

```sh
mvn clean package
```

## Configuring
Edit the mycrawler.yml file as needed.

## Running
From the project-folder:
```sh
java -jar target/MyCrawler-0.0.1-SNAPSHOT.jar mycrawler.yml
```
